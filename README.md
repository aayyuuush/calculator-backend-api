# Calculator Backend API

A production-style calculator backend built using:

- FastAPI
- Docker
- Pydantic
- Logging
- Error Handling

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Power
- Square Root

## Tech Stack

- FastAPI
- Uvicorn
- Docker
- Python 3.10
